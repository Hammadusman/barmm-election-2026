# 📊 BARMM Parliamentary Election Dashboard

A simple, interactive web-based election simulation system that calculates party votes, allocates seats, and displays real-time results with charts. Built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ➕ Add multiple political parties  
- 🗳️ Enter votes per party  
- 📊 Automatic seat allocation (40-seat system)  
- 📈 Live bar graph visualization using Chart.js  
- 🔄 Real-time summary updates  
- 💾 Auto-save data using browser storage  
- ♻️ Restore data after refresh  
- ❌ Remove party function  
- 📱 Fully responsive design (mobile, tablet, desktop)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Modern UI Design)  
- JavaScript (Vanilla JS)  
- Chart.js  
- localStorage (Browser Storage API)

## ⚙️ How to Run

1. Download or clone the project  
2. Open `index.html` in your browser  
3. Start adding parties and votes  

---

### 🌐 Option 2: Online Hosting

You can deploy this using:
- GitHub Pages  
- Netlify  

Once deployed, you will get a live link like:
```
https://yourusername.github.io/barmm-dashboard/
```

---

## 📊 How It Works

1. Add party name and votes  
2. System automatically computes:
   - Total voters  
   - Vote percentage  
   - Seat allocation (40 seats total)  
3. Click **Calculate** to generate:
   - Results table  
   - Graph visualization
